Project: ShoeScopeüëü

- A web platform dedicated to sneaker enthusiasts and custom shoe designers.

  • Enables users to post their custom shoe designs, share photos, and details about the creation process.
  • Features a user rating system, allowing the community to vote on their favorite designs.
  • Incorporates a robust search and filtering system to explore shoes based on style, color, designer, and popularity.
  • Supports user accounts with profile customization, showcasing their designs and interacting with followers.
  • Highlights include a responsive design for mobile and desktop views, ensuring accessibility and user engagement.
  • Technologies Used: React.js for the frontend, SQLite3 for the backend. Cloudinary is used for image storage and management.
  • The project emphasizes modern web development practices, including RESTful API design, security considerations with JWT authentication, and user experience design principles.
  • For more insights, including a code walkthrough and live demo, check the video above.
